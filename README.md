## **주요 기능**
- 학습놀이<br>
단어 쓰기와 단어 읽어 주기 기능을 통해 단어 학습
- 복습놀이<br>
그림에 맞는 단어를 선택하는 단어 복습 퀴즈
- 재미동요<br>
아이들이 좋아하는 핑크퐁 동요 동영상을 제공
- 나만의 단어장<br>
나만의 단어를 직접 만들어 등록하고 학습할 수 있음
